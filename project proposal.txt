BZ-Agent: Smart-City Mobility & Event Orchestrator
The objective of this project is to develop a cloud-native, multi-agent AI system that acts as an autonomous smart-city orchestrator for Bolzano. The application leverages an Agentic RAG (Retrieval-Augmented Generation) architecture to fetch and process real-time mobility, weather, and tourism data from the local Open Data Hub (ODH) APIs and other resources.
The system follows a microservices architecture, where distinct AI agents are deployed to handle specific tasks: data retrieval, constraint reasoning, and itinerary generation. By utilizing Amazon Web Services (AWS), the project aims to demonstrate how complex, multi-agent backend workflows can be efficiently orchestrated and scaled. The infrastructure relies on compute services like AWS Lambda or Amazon ECS for agent execution, Amazon DynamoDB for managing agent state and user sessions, and Amazon API Gateway for handling backend client requests.
